# Kass POS Documentation

This documentation explains the main Kass POS business flows, domain models, and system diagrams.

## Contents

- [POS](business-rules-and-processes.md) - architecture, domain models, flows, access checks, checkout, sales, master data, and inventory behavior.
- [Files](files.md) - documentation file index and diagram asset layout.

## Diagram Format

The `diagrams/` directory keeps PlantUML source files (`.puml`) and exported SVG files. Use the SVG files in published GitBook pages because they render directly in the browser.
